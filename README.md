# Student Management System

A simple Student Management System developed using Python and Django.

## Features

* Add student records
* Store student information in database
* View student details on a webpage
* Manage records using Django Admin Panel

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite

## Run the Project

```bash
python manage.py migrate
python manage.py runserver
```

Open:

http://127.0.0.1:8000/
